Phase 0 — Mindset

The only thing you need to understand before we start.

You Are the Puppet Master

You don't write code. You tell someone who does. Here's the cast:

WhoRoleWhere
You The puppet master — you decide what gets built and check the work Right here
Claude (web chat) Your assistant — helps you plan, brainstorm, and write your project brief claude.ai
Claude Code The puppet — reads your files, writes code, runs commands, does what you tell it Your terminal (CLI)

That's the whole setup. You direct. Claude builds. This guide teaches you how to direct well.

What is a CLI?

CLI stands for Command-Line Interface — a way of using software by typing text commands instead of clicking buttons. Claude Code is a CLI tool. You type instructions in a terminal window, Claude reads them and does the work.

What is "shipping"?

"Shipping" means making your work available to users. When your code goes live and people can use it, you've shipped it.
