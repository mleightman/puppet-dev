Phase 9 — Daily Skills

Screenshots, slash commands, and session management.

These are the practical skills you'll use every session. They're small individually, but together they make the difference between fighting Claude and flowing with it.

Screenshots & Images

Claude can read images. This is huge — instead of trying to describe what's wrong, you can just show it.

How to Share a Screenshot

  1. Take a screenshot: press Cmd + Shift + 4, then drag to select the area
  2. The screenshot saves to your Desktop by default
  3. Tell Claude: See the screenshot on my Desktop
  4. Claude finds it, reads it, and can diagnose the problem

Built-in Slash Commands

Claude Code has built-in commands (these are separate from your custom -commands in CLAUDE.md):

CommandWhat It DoesWhen to Use
/help Show Claude Code's built-in help When you need to find a feature
/clear Clear the conversation and start fresh When Claude is confused or stuck in a bad mental model
/compact Compress the conversation to save context space When the session is getting long but you want to keep going
/model Switch between Claude models Rarely needed — the default is usually best

Session Management

Long sessions degrade. Here's how to keep Claude sharp:

/compact — Compress the Conversation

When a session gets long, Claude starts losing track of earlier work. /compact compresses the conversation so Claude can keep going without forgetting. Use it when:

  • Claude starts repeating itself
  • Claude forgets something you discussed earlier
  • The session feels sluggish

Start Fresh

Sometimes it's better to start a new session entirely. If Claude is stuck in a bad mental model — giving wrong answers, repeating the same mistake — a fresh session clears the slate. Your CLAUDE.md reloads automatically, so Claude still knows your project.

Good time to start fresh: when you've been at it for more than an hour and things feel off.

The ! Prefix

Type ! followed by a command to run it directly from the Claude Code prompt:

! git status
! npm test
! ls -la

The output lands in the conversation, so Claude can see it too. Useful for quick checks without leaving Claude.

Escape Key — Stop Claude

Press Escape to interrupt Claude mid-stream. Use this when you see Claude going in the wrong direction — don't wait for it to finish building the wrong thing. It's like saying "stop, hold on." Claude will stop and wait for your next instruction.

Don't Worry About

  • Perfect screenshot technique — any screenshot works, Claude is flexible
  • Session length rules — just start fresh when things feel off
