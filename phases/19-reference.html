Troubleshooting — Taught in Phase 15

When This HappensDo This
Claude forgets earlier work/compact or start a fresh session
Claude goes in the wrong directionPress Escape, then redirect
Claude adds features you didn't ask for"Just fix that one issue" or point at exclusion list
Deploy failsCopy Railway error logs, paste to Claude
Something that worked stops working-status then check recent commits
Claude invents bugsNever say "find and fix bugs" — describe specific symptoms
Session feels sluggishStart a fresh session — long contexts burn tokens
CLAUDE.md didn't loadCheck for the session greeting, restart if missing

Glossary

TermDefinition
Repository (repo)A project folder on GitHub that tracks all changes
CommitA saved snapshot of your project at a point in time
BranchA separate copy of your project for making changes safely
PushUpload local changes to GitHub
PullDownload latest changes from GitHub
DeployUpload code to a server and make it available to users
Environment variableA secret your app reads at startup, stored on the server
TerminalA text-based interface for controlling your computer
CLICommand Line Interface — a tool you use by typing commands
APIA way for programs to talk to each other
ConvergenceWhen critique passes find zero actionable issues — work is clean
Scope creepWhen features get added that weren't in the original plan
PR (Pull Request)A proposal to merge code changes into the main branch
TokenUnit of AI usage — every message, file read, and tool call costs tokens
