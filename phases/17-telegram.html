Phase 17 — Telegram

Control Claude Code from your phone.

What you'll learn

  • Create a Telegram bot and connect it to Claude Code
  • Send tasks to Claude from your phone while away from the computer
  • Keep your computer awake so Claude can work remotely
  • Understand the notification rules — new messages, not edits

Send tasks to Claude from Telegram while you're away from the computer. Claude does the work, sends you updates, and you review the results from your phone.

What You Need

  • A Telegram account
  • A Telegram bot (you'll create one)
  • The telegram MCP plugin configured in Claude Code (Phase 7)
  • Bypass permissions must be ON (Phase 7) — you can't approve permission prompts from your phone
What is a Telegram bot?

A Telegram bot is an automated account on Telegram that can receive and send messages. You create one through Telegram's BotFather, get a token (a secret key), and give that token to Claude Code. Then when you send a message to your bot, Claude Code receives it and does the work.

Setup

  1. Open Telegram and search for @BotFather
  2. Send /newbot and follow the prompts to create your bot
  3. BotFather gives you a bot token — save it
  4. In Claude Code, run /telegram:configure and paste your bot token
  5. Run /telegram:access to approve your Telegram account
  6. Keep your computer awake while Claude works: 
    caffeinate -dims

    Run this in a separate terminal tab. It prevents your Mac from sleeping.

    Adjust your power settings to prevent sleep, or use a keep-awake utility.

  7. Verify bypass permissions are enabled in ~/.claude/settings.json

How It Works

  1. You send a message to your bot from Telegram
  2. Claude Code receives it and immediately acknowledges (says what it's about to do)
  3. Claude does the work — reading files, making changes, running commands
  4. Every status update is a new message (not an edit — edits don't trigger phone notifications)
  5. When done, Claude sends a completion message with elapsed time

Telegram Behavior Rules

These are taught to Claude via your CLAUDE.md:

RuleWhy
Instant ack before doing workSo you know Claude received the message
New messages only, never editEdits don't trigger push notifications
Show duration on completion (MM:SS)So you know how long it took
Narrate what you're doingYou can't see tool calls from your phone

What You Can Do from Telegram

  • Run -status and tell me what's broken
  • Plan the next feature on the todo list
  • Run -audit10 and send me the findings
  • Commit and push the current changes
  • Morning briefings sent to your phone automatically (see Phase 18: Cloud Tasks)

Don't Worry About

  • Telegram bot APIs or webhook configuration — Claude handles the setup
  • Security — access control limits who can send commands to your bot
  • Using this on day one — set it up when you want remote access
